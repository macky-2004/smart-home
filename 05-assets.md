# AI Asset Selection Policy

For every section, Antigravity AI must:

1. Understand the purpose of the section.

2. Read the copywriting before selecting an image.

3. Search only for high-quality royalty-free commercial-use assets.

4. Compare multiple suitable images.

5. Select the image that best matches the website's visual language.

6. Ensure all selected images feel like they belong to the same brand.

7. Download/organize the assets locally (prefer WebP) before integrating them into the project.

8. Never use random images simply because they are available.

9. Never mix different editing styles or inconsistent lighting.

10. Prioritize storytelling over decoration.
